# FitTrack

シンプルで爆速な筋トレ記録アプリ。ノートに書く感覚で使える、余計な機能ゼロのフィットネストラッカー。

## 特徴

- **ワンタップで前回記録を確認** — 重量・回数の迷いをなくす
- **自由メモ欄** — 体調・フォームなど何でも記録
- **広告なし・サブスクなし** — 完全無料
- **オフライン完全対応** — ネット不要

## 対応環境

- Android 8.0 以上（API 26+）

## 技術スタック

- **言語:** Kotlin
- **UI:** Jetpack Compose
- **DB:** Room (SQLite)
- **DI:** Hilt
- **グラフ:** Vico

## セットアップ

```bash
git clone https://github.com/nobu0917/fittrack-android.git
cd fittrack-android
# Android Studio で開いてそのままビルド可能
```
